<!DOCTYPE html>
define que este documento é um documento HTML5.

<html>
é o elemento raiz de uma página HTML.

<head>
contém informações meta sobre a página HTML.

<title>
especifica um título para a página HTML (que é exibido na barra de título do navegador ou na aba da página).

<body>
define o corpo do documento e é um contêiner para todo o conteúdo visível, como cabeçalhos, parágrafos, imagens, hiperlinks, tabelas, listas, etc.

<h1>
define um cabeçalho grande.

<p>
define um parágrafo.
